# Codex Handoff Pack

This directory is intended to be copied into a new project repository before the principal Codex build session begins.

Recommended order:

1. create an empty repository;
2. copy this pack into the repository root;
3. rename this file to `README-HANDOFF.md` or merge it into the project README;
4. open the repository in Codex;
5. paste `prompts/CODEX_BOOTSTRAP.md`;
6. preserve that principal session for the hackathon `/feedback` requirement;
7. respond only at the five acceptance gates.

The human owner should not approve a gate based only on screenshots. Run the specified listening and interaction steps with headphones.

Core demo story:

1. natural-language scene becomes editable geometry;
2. listener moves behind a concrete wall;
3. sound becomes quieter and darker;
4. opening a doorway redirects perceived sound toward the portal;
5. changing material/room scale changes reflections and reverb;
6. debug view explains deterministic values;
7. Raw/Simulated A/B makes the difference immediate.

Project identity and final product name are intentionally not fixed. The event guidance emphasizes that the entrant should make the naming decision.
