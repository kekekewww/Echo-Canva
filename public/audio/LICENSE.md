# Echo Canvas procedural audio assets

The WAV files in this directory are original procedural signals authored for Echo Canvas.
They are generated deterministically by `scripts/generate-audio-assets.mjs` from synthesized
tones and seeded pseudorandom noise; they contain no third-party recordings or samples.

Copyright (c) 2026 Echo Canvas contributors. Licensed for use and redistribution with this
project under the same terms as the project source. Attribution: "Echo Canvas procedural
generator, 2026."

Format: mono, 48 kHz, signed 16-bit PCM WAV. These signals are prototype fixtures, not
production sound libraries.
