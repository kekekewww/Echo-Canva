# 08 — License、Asset 與 Security Audit

Audit date: **2026-07-20**

Detailed evidence: `artifacts/release/security-license-audit.md`

## Repository

- [x] `.env`、API key、token、cookie 未追蹤；`git ls-files '.env*'` 只回報 `.env.example`。
- [x] `.env.example` 只有欄位名稱與空值。
- [x] 目前可到達 Git history 不含 `sk-proj-`／`sk-or-` credential-shaped value（只計數，不輸出環境內容）。
- [x] provider credential 只由 Node server route 使用；client source 沒有 `process.env`／`NEXT_PUBLIC` credential read。
- [x] AI routes 不記錄完整 prompt、secret 或個人資料。
- [x] `pnpm audit --audit-level critical` 通過：0 critical、0 high。
- [!] `pnpm audit` 仍報告 Next.js transitive `postcss@8.4.31` 的 1 個 moderate advisory；本產品不接受或 stringify 使用者 CSS。依 integration freeze 不升級相依套件，列入部署後依賴更新事項。

## GPT API

- [x] Prompt 最多 2,000 字元。
- [x] Compile 與 Explain 各限制每個 derived client key 每分鐘 10 次，並限制 tracked keys 數量。
- [x] 預設 30 秒 timeout，可由 server-only environment 調整。
- [x] Responses-compatible request 使用 strict JSON Schema 且 `tools: []`。
- [x] SceneSpec、Hybrid spatial metadata、labels 與幾何 domain validation 均在套用候選前執行。
- [x] 最多一次 repair；第二次仍不合法即回傳 typed failure。
- [x] Built-in assets 僅接受 `/audio/` local URL；model output 中 URL、markup、executable protocol 與 instruction-like label 會被拒絕。
- [x] 模型輸出只解析成資料，不執行程式碼或載入任意 URL。
- [x] API unavailable、timeout、refusal 與 request failure 都保留 preset/manual fallback。
