# 06 — 公開 Repository README 核對表

根目錄 `README.md` 已是產品真實來源，不應以早期 2D-only 範本覆寫。發布前應確認它持續包含：

- unified modelling workspace 與獨立 2.5D／3D cache；
- middle-button／Shift-left pan、wheel zoom、Home、Frame All；
- multiple listeners、最多四個 point sources 與本機音訊隱私界線；
- room、finite wall、Portal thickness／bottom／top 編輯；
- reversible Disable 與 Delete 的差異；
- matched-revision direct／blocked／Portal／first-order wall-floor-ceiling overlays；
- 本機啟動、optional AI provider、preset-only fallback；
- import/export 與 local-audio relink 行為；
- 完整驗證命令：`pnpm lint`、`pnpm typecheck`、`pnpm test`、`pnpm build`、`pnpm e2e`；
- architecture、Codex、GPT-5.6、assets、license、limitations；
- public demo URL（只有部署後才加入）。

## 不得重新加入的舊文案

- `The submitted version is a perceptual 2D/2.5D prototype`；目前 release candidate 已包含受限且可測試的 Hybrid 3D authoring。
- `Hybrid 3D is a future roadmap feature`；應改成 arbitrary meshes、engine export、SOFA/head tracking 等真實未完成項目。
- `pnpm test:e2e`；目前實際 script 名稱為 `pnpm e2e`。
- 未在 `.env.example` 或程式中存在的 model environment variables。
- 將 Browser HRTF 說成 MIT/KEMAR，或把 Portal path 說成 true diffraction。
